# Dataset Integration

This folder stores dataset resources linked from the frontend.

## Notes

- Some resources in `index.html` are datasets.
- Some are model or reference links, not downloadable datasets.
- For restricted datasets, placeholder folders and instructions are created so the project stays organized.
